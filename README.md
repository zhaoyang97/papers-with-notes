## 📐 Page layout

Each conference page contains:

- **Topic index**: two-level TOC across 8 categories (LLM / Generation & Multimodal / NLP Applications / Visual Perception / Decision & Embodied / Foundations & Theory / Science & Cross-Disciplinary / Others)
- **Paper entries**: multi-line format; the title line may carry inline acceptance-type badges (🎤 Oral / ✨ Spotlight / ⭐ Highlight / 📖 Findings / 🛠️ Workshop / 🏭 Industry / 🎬 Demo / 📌 Poster / 🏆 Best Paper / 🏅 Outstanding)
  ```
  **Paper Title** · 🎤 Oral · 🛠️ Workshop

  - Paper: <https://arxiv.org/abs/...>
  - Code: <https://github.com/...>
  - Notes (Chinese): [papernotes.org/…](https://papernotes.org/...)
  - Notes (English): [en.papernotes.org/…](https://en.papernotes.org/...)
  ```
